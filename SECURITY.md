# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | Yes       |
| 0.1.x   | No        |

## Responsible Use

This tool is intended for:
- Scanning your own codebases and repositories
- Security audits with explicit authorization
- Responsible disclosure workflows

Do not use this tool to scan repositories you do not own or have permission to audit.

## Reporting a Vulnerability

If you find a security issue in secret-scanner itself, please open a private security advisory at:

https://github.com/Vasishta03/secret-scanner/security/advisories/new

Include a description of the issue, steps to reproduce, and potential impact. You will receive a response within 48 hours.

Do not open a public issue for security vulnerabilities.
